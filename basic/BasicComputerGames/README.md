These are programs from the book "Basic Computer Games Microcomputer
Edition" by David H. Ahl, 1978.

They have been adapted to run on the 68000 version of Enhanced Basic.

Most games have instructions in the .txt file corresponding to the
.bas file for the game.

Some games were omitted as they were not easy to port.

Git normally converts the line ending characters for text files to
whatever your platform uses when you do a git clone, although this is
configurable. Enhanced Basic uses Carriage Return to separate lines.
Depending on the platform you are using, you may need to convert the
line endings or use a file transfer program that does this when you
transfer files over a serial port. I use the ascii-xfer program under
Linux. The Linux unix2dos and dos2unix programs can also be used to
convert files.
